О Vault

Пет-проект, который вырос из лабы по контейнерам и облакам — и которым я пользуюсь сам.

Что делает

Четыре вещи, обычно разбросанные по разным сайтам сомнительного качества.

Зачем я это делал

  • Надоело гуглить «генератор паролей».
  • Хотелось на живом примере разобраться с Fernet и HIBP k-anonymity.
  • Нужно было что-то сдавать за лабу, но чтобы результат был полезный.

Под капотом

  • Python / Flask + ванильный JS.
  • PostgreSQL в Docker, только зашифрованные токены, TTL 24 часа.
  • docker-compose поднимает web + db одной командой.
  • Render + GitHub Actions = автодеплой на каждый push в main.
  • Метрики через prometheus-flask-exporter.
  • Kubernetes-манифесты в репо, готовы к применению.

Чего здесь нет

  • Регистрации и аккаунтов.
  • Логов с паролями и plaintext.
  • Небезопасных алгоритмов.
  • Рекламы, трекеров, cookie-banner'ов.

Немного статистики

— токенов зашифровано
